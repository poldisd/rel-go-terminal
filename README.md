# REL<GO> — Relationship Map Terminal

Bloomberg-Terminal-Style Beziehungsnetzwerk-Explorer (Holders, Board, Banks, Kunden, Lieferanten) als reines HTML/JS/D3-Frontend.

## Deploy auf Vercel
1. Dieses Repo zu deinem eigenen GitHub-Account pushen (siehe unten)
2. Auf vercel.com → "Add New Project" → dieses Repo auswählen → Deploy
   (kein Build-Step nötig, ist eine statische index.html)

## Lokal öffnen
Einfach `index.html` im Browser öffnen.
